

Assignment Project :
Employee Management Dashboard

Project Overview : 
This project is developed as part of a company assignment to demonstrate practical understanding of frontend development concepts and proper project setup.

The application is built with a clean structure and focuses on usability, readability, and maintainability. It follows standard development practices and can be easily run on a local system.

---


Tech:
- React
- Vite
- LocalStorage

---
Steps to Run the Project

1. Clone the Repository
git clone https://github.com/Prammasani-Ajay/Assignment-EMPLOYEE-DASHBOARD.git

Steps:
1. npm install
2. npm run dev
3. Open in Browser ( http://localhost:5173/)

Features:
- Login
- CRUD Employees
- Image upload preview
- Print list
